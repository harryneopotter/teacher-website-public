# Favicon Information

The favicon.svg file contains a custom-designed icon that matches the creative writing theme:

## Design Elements:
- Purple to pink gradient background (matches site theme)
- White document/book icon representing creative writing
- Text lines showing written content
- Golden pen tip representing the writing process
- Rounded corners for modern appearance

## Colors Used:
- Purple: #8B5CF6 (violet-500)
- Pink: #EC4899 (pink-500) 
- Gold: #F59E0B (amber-500)
- White: #FFFFFF

## Browser Support:
- Modern browsers will use the SVG favicon
- Fallback to generic icon for older browsers

To create additional formats:
1. Convert favicon.svg to PNG at 32x32, 16x16 sizes
2. Use online tools to create favicon.ico from PNG files
3. Add apple-touch-icon.png (180x180) for iOS devices
